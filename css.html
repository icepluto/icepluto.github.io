派生选择器：通过依据元素在其位置的上下文关系来定义样式，你可以使标记更加简洁

      ul li{
        font:10px;
      }
后代选择器：不论嵌套有多深都可以找到，例如： 
      
        ul em{
          font:red;
        }
      
    
      <ul>
        <li>
          <em>
            hello
          <em>
        <li>
      <ul>
子元素选择器： u l > l i ;
相邻兄弟选择器： div + p
属性选择器：[title]
